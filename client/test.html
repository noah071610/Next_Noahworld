Html style 과 scss, styled-components에 글로벌스타일&미디어쿼리, atnd 등등 여러 가지 스타일시트나 라이브러리가 복잡하게 얽혔습니다.

이번 프로젝트에서 가장 후회하고 절대로 반복해선 안될 실수라 생각합니다.

추후에는 체계적으로 변수이름을 짓고, styled-component을 필두로 미디어 쿼리까지 작성하며 공통적용 스타일은 scss로 사용 할 생각힙니다.


계획과는 다른 프로젝트가 됐다.


처음에는 간단한 포트폴리오를 만들 생각이었으나 블로그 개설에 욕심이 생겨 도중에 포트폴리오 페이지를 컴포넌트로 빼버리고 블로그를 메인으로 작업했습니다. 

Javascript -> React -> NEXT 순으로 계속 변화하며 변수명과 컴포넌트가 얽히고, 똑같은 작업을 여러번 반복한 감이 없지않아 있습니다. 

요구사항을 정확하게 인식하고 초반부에 아키텍처를 체계적으로 정리하는 중요성을 절실히 깨달았습니다.
